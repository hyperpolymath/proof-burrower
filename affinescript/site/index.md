# AffineScript

The public web home for this project is [affinescript.dev](https://affinescript.dev).

The language where protocol correctness is free.

AffineScript is the only language that combines affine types, quantitative type theory, row polymorphism, and algebraic effects in a single practical systems language. This intersection makes it possible to write code where the compiler proves your protocol states, resource lifecycles, and effect boundaries are correct -- with syntax that doesn't feel like a theorem prover.
